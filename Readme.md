# FUTURE_CS_02 – SOC Monitoring & Incident Response

## Overview
This project simulates the core responsibilities of a Security Operations Center (SOC) analyst.  
Security events were analyzed using Elastic Security to identify threats and respond appropriately.

## Tools Used
- Elastic Cloud (Elasticsearch)
- Kibana (Elastic Security)
- Dev Tools (manual event ingestion)

## Activities Performed
- Security event analysis
- Alert severity classification
- Incident identification
- Incident response planning

## Key Incidents Identified
- Brute force login attack
- Malware detection on endpoint
- Suspicious login behavior

## Learning Outcomes
- Understanding SIEM dashboards
- SOC alert triage
- Incident escalation process
- Real-world SOC workflow simulation
